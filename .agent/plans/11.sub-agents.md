# Module 8: Sub-Agents

## Context

The app currently has a multi-tool agent (Module 7) that can call `search_documents`, `query_structured_data`, and `web_search` — but always does cross-document retrieval. When a user asks "summarize report.pdf" or "what are the key findings in the quarterly review?", the agent does a semantic search and returns a few chunks, which is insufficient for full-document analysis.

Module 8 adds **sub-agents**: the main agent detects full-document scenarios, spawns an isolated Gemini call with all of that document's chunks loaded into context, and streams the sub-agent's reasoning to the UI as a collapsible nested section.

## Task 2: Backend — Sub-agent service

**File:** `backend/app/services/sub_agent.py` (NEW)

Core function:

```python
@traceable(name="sub_agent_analyze", run_type="chain")
def run_sub_agent(
    document_id: str,
    document_name: str,
    question: str,
    user_id: str,
    supabase_client,
) -> Generator[tuple[str, str], None, None]:
```

Yields event tuples:
- `("sub_agent_start", json.dumps({"document_name": doc_name}))`
- `("sub_agent_token", chunk_text)` — streamed answer tokens
- `("sub_agent_done", full_result_text)`

**Implementation:**
1. Load ALL chunks for `document_id` ordered by `chunk_index` from `document_chunks`
2. Concatenate into a single context string
3. If total chars > 800K, truncate with a note (Gemini context limit safety)
4. Build a focused system prompt: "You are a document analyst. Below is the full content of '{document_name}'. Analyze it to answer the user's question thoroughly."
5. Inject full document content into system prompt
6. Single streaming Gemini call (no tools needed — the sub-agent has the full document in context already, so no tool decision call required)
7. Yield `sub_agent_start`, then `sub_agent_token` for each chunk, then `sub_agent_done` with full result

**Key:** The sub-agent does NOT need the hybrid two-call pattern because it doesn't use tools — it has the full document already. Just one streaming call.

Reuses `_get_client()` and `get_llm_model()` from `openai_client.py`.

**Validation:** Call with a known document_id, verify yields start → tokens → done with non-empty content.

```python
elif tool_name == "analyze_document":
    from app.services.sub_agent import run_sub_agent
    doc_name = args.get("document_name", "")
    question = args.get("question", "")

    # Resolve document_name → document_id via fuzzy match
    doc = supabase_client.table("documents") \
        .select("id, original_filename") \
        .eq("user_id", user_id) \
        .ilike("original_filename", f"%{doc_name}%") \
        .order("created_at", desc=True) \
        .limit(1).execute()

    if not doc.data:
        result_text = f"No document matching '{doc_name}' found."
    else:
        doc_id = doc.data[0]["id"]
        actual_name = doc.data[0]["original_filename"]
        sub_agent_result = ""
        for evt_type, evt_data in run_sub_agent(doc_id, actual_name, question, user_id, supabase_client):
            yield (evt_type, evt_data)  # Pass sub-agent events through
            if evt_type == "sub_agent_done":
                sub_agent_result = evt_data
        result_text = sub_agent_result
```

### messages.py — `event_generator()` changes

Forward new event types to client and track tool metadata:

```python
def event_generator():
    full_response = ""
    tool_metadata = None

    for event_type, data in stream_response(...):
        if event_type == "token":
            full_response += data
            yield json.dumps({"type": "token", "content": data})
        elif event_type == "sub_agent_start":
            tool_metadata = {"tools_used": [json.loads(data)]}
            yield json.dumps({"type": "sub_agent_start", **json.loads(data)})
        elif event_type == "sub_agent_token":
            yield json.dumps({"type": "sub_agent_token", "content": data})
        elif event_type == "sub_agent_done":
            if tool_metadata and tool_metadata["tools_used"]:
                tool_metadata["tools_used"][0]["tool"] = "analyze_document"
                tool_metadata["tools_used"][0]["sub_agent_result"] = data[:300]
            yield json.dumps({"type": "sub_agent_done"})
        elif event_type == "done":
            yield json.dumps({"type": "done"})

    insert_data = {
        "thread_id": thread_id, "user_id": user_id,
        "role": "assistant", "content": full_response,
    }
    if tool_metadata:
        insert_data["tool_metadata"] = json.dumps(tool_metadata)
    supabase.table("messages").insert(insert_data).execute()
```
