<objective>
Extend `GET /api/folders` so the `subfolders` array returns objects shaped `{id: UUID | null, path: string}` instead of bare path strings. Frontend Plan 06-05 (typed client) and Plan 06-06 (FolderNode recursion) cannot wire `DELETE /api/folders/{id}` or `PATCH /api/folders/{id}` for pre-existing folders without a UUID round-tripped on the list call.

Purpose: Closes D-06 (the plan-checker blocker on Plan 06-09 — folder-id resolution gap). Cleanest contract: one round-trip, no separate path→id lookup endpoint. Inferred-only subfolders (folders with no explicit `folders` row, only documents at descendant paths) carry `id: null` so the frontend can disable rename/delete affordances on them.

Output:
- `folder_service.list_folder` returns `subfolders: List[{id: UUID | None, path: str}]`.
- `FolderRef` + `FolderListResponse` Pydantic models in `schemas.py`.
- `routers/folders.py` declares `response_model=FolderListResponse` on the GET endpoint.
- New pytest verifying response shape on a nested folder.
</objective>

import re
from test_helpers import h, sign_in, base_url, cleanup_folder

UUID_RE = re.compile(r"^[0-9a-f]{8}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{12}$", re.I)

def test_subfolders_carry_id():
    token = sign_in("test@test.com")
    headers = {"Authorization": f"Bearer {token}"}

    # Create parent + child
    parent = h.post(f"{base_url}/api/folders", json={"path": "/d06-test-parent", "scope": "user"}, headers=headers).json()
    child = h.post(f"{base_url}/api/folders", json={"path": "/d06-test-parent/child", "scope": "user"}, headers=headers).json()

    try:
        # GET parent's contents
        res = h.get(f"{base_url}/api/folders?path=/d06-test-parent&scope=user", headers=headers).json()
        h.test("subfolders is a list", isinstance(res["subfolders"], list) and len(res["subfolders"]) >= 1)

        h.test("subfolder has path key", "path" in first)
        h.test("subfolder id is a UUID string", isinstance(first["id"], str) and UUID_RE.match(first["id"]))
        h.test("subfolder path matches", first["path"] == "/d06-test-parent/child")
    finally:
        # Cleanup ONLY the IDs we created (CLAUDE.md mandatory rule)
        cleanup_folder(child["id"], headers)
        cleanup_folder(parent["id"], headers)

if __name__ == "__main__":
    test_subfolders_carry_id()
    print("OK: D-06 subfolder id test passed")
